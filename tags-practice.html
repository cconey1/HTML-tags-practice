Headings

Lists

Unordered Lists

Unordered lists are marked with bullet pointsand go in 'ul' tags. Each item goes in 'li' tags.

Types of Lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered lists

Orderedd lists are numbered and go in 'ol' tags. Each list item goes in an 'li' tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to DuckDuckGo to search for more information

Text Emphasis

Italics

the 'em' element adds emphasis to the text within the opening and closing tags. This shows up as Italics in most browsers.

Bold

Use the 'strong' element to make things bold.
